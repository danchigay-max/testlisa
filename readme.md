# Кошкин дом — викторина по сказке С. Я. Маршака

Интерактивный тест для детей с озвучкой, анимациями и доступным интерфейсом для незрячих и слабовидящих.

## Структура

- 3 раунда по 3 вопроса
- Между раундами — объявление с озвучкой
- После прохождения — поздравление

## Что внутри

- `index.html` — главная страница с экраном выбора теста, приветствия, раундов, вопросов и результата
- `styles.css` — стили, адаптив, акценты для слабовидящих
- `script.js` — логика раундов, ответов и воспроизведения MP3
- `audio/koshkin-dom/` — аудиофайлы для вопросов, ответов, раундов и озвучки
- `images/` — обложка и картинки героев

## Озвучка

Каждый вопрос и каждый ответ озвучиваются отдельным MP3-файлом. Файлы лежат в `audio/koshkin-dom/`.

## Деплой

Статический сайт. Можно разместить на любом веб-сервере (Nginx, Apache, GitHub Pages и т.д.).
