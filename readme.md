# Сказочный тест — рабочий макет

Статический прототип детского квиза с раундами, озвучкой через MP3 и возможностью повторного ответа при ошибке.

## Что внутри

- `index.html` — главная страница, экраны выбора теста, раунда, вопросов и результата
- `styles.css` — стили, адаптив и акценты для слабовидящих
- `script.js` — логика раундов, ответов и проигрывания MP3
- `audio/` — папка для аудиофайлов
- `images/` — папка для изображений (герои по краям)

## Озвучка

Озвучка ожидается через MP3-файлы. Структура для первого теста:
- `audio/koshkin-dom/round-1.mp3`
- `audio/koshkin-dom/round-2.mp3`
- `audio/koshkin-dom/round-3.mp3`
- `audio/koshkin-dom/correct.mp3`
- `audio/koshkin-dom/wrong.mp3`
- `audio/koshkin-dom/complete.mp3`
- `audio/koshkin-dom/r1q1-question.mp3` и т.д. для каждого вопроса и ответа

## Картинки

Для героев по краям экрана используй:

- `images/hero-right.png`

## Деплой

Можно размещать как статический сайт. Для git-деплоя на сервер по IP и порту достаточно bare-репозитория и веб-сервера.
Когда будешь готов — опиши сервер (Nginx/Apache, путь к webroot), и я дам точные шаги.
